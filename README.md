# ubuntu-mirrors-change

快捷修改ubuntu系统apt使用的镜像源，安装docker-ce。

## 使用方法

bash <(curl -Ls https://raw.githubusercontent.com/qjds/mirrors-change/main/ubuntu.sh)

### 目前已添加的镜像源：

* 中科大
* 阿里云
* 自定义源替换

#### 一键安装docker

### 已添加容器镜像

* minio 
* 自定义容器镜像

#### netplan网络管理器修改dhcp为静态

#### 修改ssh配置允许root登录
